Step Game Path

Step Game is A game played with a dice.

Your the dice one and target the number on the play ground.

After rolling you Select a number and click on it.

The the hexagon containing the number place will get colored.

The number of dice is a random number to choose from play ground number.

The number of Dice is the number of steps.

Completing one path will win.

Path starts from the lower row to the top row.
